    Class HashMap<K, V>

    HashMap - A hash table implementation with separate chaining for collision resolution. Maintains insertion order and automatically rehashes when load factor is exceeded. Uses FNV-1a hash function for string keys.

    Type Parameters

    • K

      The type of keys (must be convertible to string)

    • V

      The type of values
