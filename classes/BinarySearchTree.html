    Class BinarySearchTree<T>

    Binary Search Tree - A tree structure where each node's left children are smaller and right children are larger than the node's value. Provides O(log n) average time for search, insert, and delete operations.

    Type Parameters

    • T

      The type of values stored in the tree (must be comparable)
