    Class PriorityQueue<T>

    PriorityQueue - A queue where elements are dequeued based on priority. Built on top of a Heap for efficient O(log n) operations. Lower priority numbers are dequeued first by default.

    Type Parameters

    • T

      The type of values stored in the queue

        Optional comparison function for priorities. Default: (a, b) => a - b (lower priority first)

      Returns PriorityQueue<T>

      Example

      const pq = new PriorityQueue<string>();
      pq.offer('low priority task', 5);
      pq.offer('high priority task', 1);
      console.log(pq.poll()); // 'high priority task'
