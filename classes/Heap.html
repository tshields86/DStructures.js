    Class Heap<T>

    Heap - A complete binary tree data structure that satisfies the heap property. Can be configured as min-heap or max-heap via comparison function. Provides O(log n) insertion and O(1) access to min/max element.

      The type of elements stored in the heap

      Parameters

      • compareFn: CompareFn<T> = ...

        Comparison function (a, b) => number. Return negative if a < b, positive if a > b, zero if equal. Default: (a, b) => a - b (min-heap)

      Returns Heap<T>

      Example

      // Min heap
      const minHeap = new Heap<number>((a, b) => a - b);
      // Max heap
      const maxHeap = new Heap<number>((a, b) => b - a);
