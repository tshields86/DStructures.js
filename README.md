## Installation

Clone the repo or install the code via NPM:

```sh
npm install structures.js
```

Then you can import it into your programs:

```js
const {
  BinarySearchTree,
  DoublyLinkedList,
  Graph,
  HashMap,
  HashSet,
  Heap,
  LinkedList,
  MaxHeap,
  MinHeap
  PriorityQueue,
  Queue,
  Stack
} = require('structures.js');
```
