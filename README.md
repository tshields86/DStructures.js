## Installation

Clone the repo or install the code via NPM:

```sh
npm install DStructures.js
```

Then you can import it into your programs:

```js
const {
  BinarySearchTree,
  DoublyLinkedList,
  Graph,
  HashMap,
  HashSet,
  Heap,
  LinkedList,
  MaxHeap,
  MinHeap,
  PriorityQueue,
  Stack,
  Queue,
  binarySearch,
  bubbleSort,
  dijkstra,
  insertionSort,
  mergeSort,
  quickSort,
  selectionSort,
} = require('DStructures.js');
```
