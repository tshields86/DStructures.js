    Function dijkstra

    • dijkstra<T>(graph: Graph<T>, startValue: T): DijkstraResult<T>

      Dijkstra's Shortest Path Algorithm - Finds the shortest paths from a source vertex to all other vertices in a weighted graph. Time complexity: O((V + E) log V) with priority queue Space complexity: O(V)

      Type Parameters

      • T

        The type of values stored in graph vertices

      Parameters

      • graph: Graph<T>

        The graph to search

      • startValue: T

        The starting vertex value

      Returns DijkstraResult<T>

      Object containing distances and previous vertices maps

      Example

      const graph = new Graph<string>();
      graph.addEdge('A', 'B', 4);
      graph.addEdge('A', 'C', 2);
      graph.addEdge('B', 'C', 1);
      graph.addEdge('B', 'D', 5);
      graph.addEdge('C', 'D', 8);

      const result = dijkstra(graph, 'A');
      const pathToD = getShortestPath(result, graph.getVertex('D')!);
      console.log(pathToD.map(v => v.value)); // ['A', 'C', 'B', 'D']
